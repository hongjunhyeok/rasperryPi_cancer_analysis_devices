----------------
Version Settings.

1. QT Version 4 

2. OpenCV 3.4.0

3. Ubuntu 16.04

4. Raspberry Pi 3

## Main Code Files
--------------
1. testprocess.pro 
- 프로그램 라이브러리 추가 및 profile 설정하는 파일

2. main.cpp
- QT 어플리케이션을 실행시켜 윈도우를 출력하는 파일

3. mainwindow.cpp
- 프로그램 핵심파트로 평준화알고리즘, 데이터베이스 연동, 이미지처리 및 GPIO 연동
